# [Dinner and a Show](https://matmcw.github.io/wdd130-patten/)

WDD 130 final project - a website for a podcast called Dinner and a Show.

| CLIENT          | LEAD               | JR        |
| --------------- | ------------------ | --------- |
| Jonathan Patten | Matthew McWilliams | Colt Rose |

## Purpose

The site serves as the central hub for the podcast and its community. It provides access to episodes, host bios, platform links, and more.

## Audience

The site targets college students who enjoy personality-driven entertainment podcasts. It's directed towards those who are active on social media and looking for humor, insight, and community.
